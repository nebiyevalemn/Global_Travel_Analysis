# Global Travel Cost Analysis Dashboard 🌍✈️💰

## 📌 Project Overview
This project is an end-to-end data analytics solution focused on analyzing travel costs across multiple countries and cities worldwide.

The goal is to help users understand and compare travel expenses by analyzing restaurant prices, hotel costs, tourist attractions, and flight ticket prices from Baku to selected destinations.

---

## ⚙️ Project Workflow

### 1. Data Collection (Python)
- Scraped data from online sources
- Collected data for:
  - Restaurant prices
  - Hotel prices
  - Tourist attraction costs
  - Flight ticket prices (Baku → selected cities, 2026)

### 2. Data Processing (Python)
- Cleaned and transformed raw data
- Standardized and normalized datasets
- Structured data into relational format

### 3. Database Integration (SQL)
- Loaded cleaned datasets into SQL database
- Designed relational tables
- Wrote analytical queries to extract insights
- Prepared final tables for visualization

### 4. Data Visualization (Power BI)
- Built an interactive dashboard
- Compared travel costs across cities and countries
- Created filters for city, country, and expense type
- Delivered final insights for decision-making

---

## 📊 Dashboard Features
- Travel cost comparison across 100+ cities
- Category-based expense analysis (food, hotel, flights, attractions)
- Interactive filtering by location
- Identification of affordable vs expensive destinations
- Final insights for travel planning

---

## 🛠️ Technologies Used
